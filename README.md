# lofi-stream-dlive

24/7 lofi stream to DLive with a space station theme.

## Live Stream

Watch at: [dlive.tv/lofistream](https://dlive.tv/lofistream) (update with actual channel)

## Theme: Space Station

An orbital space station viewport looking down at Earth, with solar panels, control panels, and ethereal space ambient audio.

## Quick Start

```bash
# View locally
cd docs && python3 -m http.server 8080
# Open http://localhost:8080
```
